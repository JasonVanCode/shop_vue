## 安装步骤

```
git clone https://github.com/JasonVanCode/shop_vue.git// 把模板下载到本地

cd shop_vue    // 进入模板目录

npm install         // 安装项目依赖，等待安装完成之后，安装失败可用 cnpm 或 yarn

//通过vue.config.js 修改端口

// 开启服务器，浏览器访问 http://localhost:3000
npm run dev

// 执行构建命令，生成的dist文件夹放在服务器下即可访问
npm run build

```
